# Android App 開源開發

![](cover.jpg)

Android App 以開源角度來開發。

前端：

* RxJava, RxAndroid
* retrolambda (java7 + java8)
* retrofit
* okhttp
* DI: Dagger2
* AutoParcel, AutoValue, etc.
* ImageLoader: fresco, AUIL, picasso, glide, etc.
* json2pojo: jackson, gson, logansquare, etc.
* Orm: DBFlow, etc.
* SimpleFacebook

後端：

* Parse
* auto-parse
* rxparse
 
開發環境：

* Android Studio
* gradle

專案環境：

* gitlab SCM
* phabricator PM
